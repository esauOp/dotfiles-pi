## Getting Started

1. Clone the project from GitHub:

  ```
  git clone https://github.com/esauOp/dotfiles-pi.git ~/dotfiles
  cd ~/dotfiles
  ```

2. Run the install script:

  ```
  sh ./install.sh
  ```

3. Run the setup script:
  ```
  sh ./setup.sh
  ```

4. Change your shell
  ```
  chsh -s /bin/zsh
  ```
